The Focuh Chrome Extension blocks distracting websites while you're in a focus session.

## Installation

### From Chrome Web Store

1. Visit the [Focuh Chrome Extension](https://chrome.google.com/webstore) page
2. Click **Add to Chrome**
3. Confirm the installation

### Manual Installation

1. Download the extension from your Focuh dashboard
2. Open `chrome://extensions` in your browser
3. Enable **Developer mode**
4. Click **Load unpacked** and select the extension folder

## Setup

After installation:

1. Click the Focuh icon in your browser toolbar
2. Click **Sign In** and authenticate with your Focuh account
3. Add websites to your block list

## Features

### Automatic Blocking

When you start a focus session in Focuh, the extension automatically blocks your configured sites.

### Focus Status

The extension icon shows your current focus status:

- **Gray** - Not focusing
- **Orange** - In a focus session

### Quick Add

Right-click on any page and select **Block this site** to quickly add it to your block list.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Sites aren't being blocked">
    Make sure you're signed in to the extension and have an active focus session running.
  </Accordion>
  <Accordion title="Extension isn't connecting">
    Try signing out and back in. Make sure you're using the same account as the web app.
  </Accordion>
</AccordionGroup>
